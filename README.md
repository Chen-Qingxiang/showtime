# Showtime Timeline

一个面向**历史 / 文学 / 地质 / 宇宙演化**等跨尺度研究场景的可视化时间轴工具。  
你可以把不同来源的 CSV 作为不同图层叠加，对同一问题进行“宏观到微观”的联动观察。

---

## 核心能力

- **多图层时间轴**：每个 CSV 自动成为一个 layer，可拖拽调整顺序、右键删除。  
- **多尺度缩放**：支持从近现代历史到地质史、宇宙史的连续缩放。  
- **双数据入口**：既可上传本地 CSV，也可直接加载仓库内置示例。  
- **灵活时间解析**：支持年份、日期、BCE/CE、区间、单点时间，兼容多种分隔符。
- **背景层库**：内置通用历史、思想、宗教、科技、经济、文学艺术和地质生命背景层，可按当前数据推荐加载。
- **可读性优化**：大数字年份会自动以 `万 / 亿` 显示，减少长数字阅读负担。

## 快速开始

1. 打开页面后，可先点击 **加载左侧文本** 体验默认数据。  
2. 通过顶部 **内置示例** 选择器，可一键加载不同学科的数据层。  
3. 通过顶部 **背景库** 选择器，可叠加通用参照层；也可以点击 **加载推荐背景**，让工具按当前主题和时间范围自动选几层。
4. 也可上传本地 CSV（两列：`time,title`），文件名会作为图层名。

## CSV 格式说明

Showtime 的 CSV 只需要两列：第一列是时间，第二列是标题。

```csv
time,title
1949-10-01,中华人民共和国成立
1914-07-28~1918-11-11,第一次世界大战
```

实际导入时，表头可以保留，也可以省略。以 `#` 开头的行会被忽略，适合写注释。

### 列定义

| 列 | 必填 | 说明 |
| --- | --- | --- |
| `time` | 是 | 年份、日期、单点时间或时间区间 |
| `title` | 是 | 事件标题，会显示在时间轴上 |

图层名不写在 CSV 里：上传文件时使用文件名作为图层名；从左侧文本框加载时使用 `文本` 图层；背景库会使用 `背景：xxx` 图层名。

### `time` 的写法

| 写法 | 示例 | 说明 |
| --- | --- | --- |
| 公元年份 | `1949` | 单点年份 |
| BCE 年份 | `-221`、`221BC`、`公元前221` | BCE 会按无公元 0 年处理，`221BC` 等价于内部值 `-220` |
| 公元日期 | `1949-10-01` | 日期格式为 `YYYY-MM-DD`，目前日期只支持公元年份 |
| 年份区间 | `1368~1644` | 从开始年份到结束年份 |
| 日期区间 | `1914-07-28~1918-11-11` | 精确到日的区间 |
| 开区间 | `1949~`、`1949-10-01~` | 自动补到当前年份或当前日期 |

推荐用 `~` 分隔区间。工具也兼容 `-`、`—`、`–`、`－`、`〜`、`～`、`至`、`到`，但在生成 CSV 时统一用 `~` 最不容易出错。

### 标题与逗号

CSV 用逗号分列。如果标题里有逗号，请用英文双引号包起来：

```csv
1942,"《春望》, 杜甫相关研究节点"
```

如果标题里有双引号，用两个双引号转义：

```csv
1967,"论文 ""The Medium is the Message"" 出版"
```

### 可复制模板

年份版：

```csv
# time,title
-221~-207,秦
1368~1644,明
1949~,中华人民共和国
```

日期版：

```csv
# time,title
1949-10-01,中华人民共和国成立
1914-07-28~1918-11-11,第一次世界大战
```

跨尺度背景版：

```csv
# time,title
-13800000000~-13700000000,宇宙大爆炸与暴涨阶段
-541000000~-485000000,寒武纪
-300000~-200000,早期智人出现
-221~-207,秦帝国
1969-07-20,阿波罗11号登月
```

常见问题：

- 不要用中文逗号 `，` 分隔列；列分隔符必须是英文逗号 `,`。
- BCE 日期暂不支持 `-221-01-01` 这类写法；BCE 请用年份。
- Excel 可能会自动改日期格式，建议导出后用文本编辑器检查是否仍是 `YYYY-MM-DD`。
- 如果某行没有显示，通常是 `time` 无法解析；先用上面的模板试一行最小样例。

## 交互操作

- 滚轮：缩放  
- 拖拽：平移  
- **Shift + 滚轮**：围绕视图中心缩放  
- **Ctrl/Cmd + 滚轮**：加速缩放  
- 左侧图层名区域拖动：调整图层顺序  
- 左侧图层名区域右键：删除图层  
- 双击画布 / 点击“重置视图”：快速回到合适视角

## 内置示例

`examples/` 目录提供了多组可直接加载的数据，包括：

- 中国朝代与皇帝在位
- 日期精度版皇帝在位与哲学家生卒年表
- 人类史与文明关键节点
- 地质年代与生命演化
- 宇宙与太阳系演化
- 文学与思想史

`background/` 目录提供 40 个通用背景层，大致分为：

- 长尺度：Big History、地质生命史、人类迁徙与人口、农业与驯化、气候环境事件
- 通用政治史：世界史分期、世界帝国、主要战争、中国朝代、欧洲时代、殖民主义与去殖民化、政治革命、民主化、法律与人权
- 区域背景：伊斯兰世界、南亚、日本、朝鲜半岛、东南亚、非洲、美洲、欧亚草原、古代近东与埃及、大洋洲与太平洋世界
- 知识文化：思想史运动、宗教史、文学艺术、教育与知识机构、文字媒介、语言文字与翻译、医学与公共卫生、天文学与空间探索
- 社会技术经济：科学技术史、贸易网络、经济社会史、资本主义与金融体系、交通运输、能源史、城市化、劳动制度 / 社会运动 / 性别

## 项目定位

Showtime Timeline 并非只服务某一条历史线，而是为了支持“**不同时间颗粒度 + 不同学科图层**”的并行观察：  
你可以在同一画面中同时查看宇宙史、地质史、生物演化与人类文明事件，快速建立跨尺度关联。
