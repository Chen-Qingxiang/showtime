# Showtime Timeline

一个面向**历史 / 文学 / 地质 / 宇宙演化**等跨尺度研究场景的可视化时间轴工具。  
你可以把不同来源的 CSV 作为不同图层叠加，对同一问题进行“宏观到微观”的联动观察。

---

## 核心能力

- **多图层时间轴**：每个 CSV 自动成为一个 layer，可拖拽调整顺序、右键删除。  
- **多尺度缩放**：支持从近现代历史到地质史、宇宙史的连续缩放。  
- **双数据入口**：既可上传本地 CSV，也可直接加载仓库内置示例。  
- **灵活时间解析**：支持 BCE/CE、区间、单点时间，兼容多种分隔符。  
- **可读性优化**：大数字年份会自动以 `万 / 亿` 显示，减少长数字阅读负担。

## 快速开始

1. 打开页面后，可先点击 **加载左侧文本** 体验默认数据。  
2. 通过顶部 **内置示例** 选择器，可一键加载不同学科的数据层。  
3. 也可上传本地 CSV（两列：`time,title`），文件名会作为图层名。  

---

## CSV 格式说明

- 基本格式：`time,title`  
- 区间：`start~end`（也兼容 `- — – － 〜 ～ 至 到`）  
- BCE：支持 `-221`、`221BC`、`公元前221`  
- 单点：`1949` 会自动按 `1949~1949` 处理  
- 开区间：如 `1949~` 会自动补到当前年份  

示例：

```csv
-221~-207,秦
1368~1644,明
1949~,中华人民共和国
```

## 交互操作

- 滚轮：缩放  
- 拖拽：平移  
- **Shift + 滚轮**：围绕视图中心缩放  
- **Ctrl/Cmd + 滚轮**：加速缩放  
- 左侧图层名区域拖动：调整图层顺序  
- 左侧图层名区域右键：删除图层  
- 双击画布 / 点击“重置视图”：快速回到合适视角

## 项目定位

Showtime Timeline 并非只服务某一条历史线，而是为了支持“**不同时间颗粒度 + 不同学科图层**”的并行观察：  
你可以在同一画面中同时查看宇宙史、地质史、生物演化与人类文明事件，快速建立跨尺度关联。
