# Showtime Background Library

这一目录放通用背景层。它们的目的不是替代专业年表，而是在用户导入一个研究主题后，快速提供可比较的历史、思想、区域、技术和长尺度背景。

CSV 格式仍然是 `time,title`。时间可以是年份、区间，或当前工具支持的公元日期 `YYYY-MM-DD`。

这些背景层适合先作为研究参照使用；精确到具体日期的事件仍建议在专题 CSV 中单独维护。

当前本地库包含 40 个背景层：

- 长尺度：Big History、地质生命史、人类迁徙与人口、农业与驯化、气候环境事件
- 通用政治史：中国朝代、世界帝国、欧洲时代、世界史分期、主要战争、殖民主义与去殖民化、政治革命、民主化、法律与人权
- 区域背景：伊斯兰世界、南亚、日本、朝鲜半岛、东南亚、非洲、美洲、欧亚草原、古代近东与埃及、大洋洲与太平洋世界
- 知识文化：思想史运动、宗教史、文学艺术、教育与知识机构、文字媒介、语言文字与翻译、医学与公共卫生、天文学与空间探索
- 社会技术经济：科学技术史、贸易网络、经济社会史、资本主义与金融体系、交通运输、能源史、城市化、劳动制度 / 社会运动 / 性别

## 选层参考框架

这些层的取舍优先服务“研究背景”而不是单一学科完整性。扩展或校订时可优先参考：

- College Board AP World History: Modern 的全球史主题和 1200 年以来的常用单元框架
- International Commission on Stratigraphy 的国际地质年代表
- Our World in Data 的人口、经济、能源、健康、技术等长时段数据项目
- WHO 的公共卫生里程碑
- NASA 的空间探索和月球探索时间线
- IEA 的能源安全与能源转型历史说明
