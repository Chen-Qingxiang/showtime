# 搜索、过滤和快捷键

## 搜索语法

搜索默认采用简单、可解释的 AND 语义：所有关键词都必须出现在同一事件的标题、图层、说明或元数据中。

```text
苏轼 密州
乌台诗案
"前赤壁赋"
1080~1085
苏轼 1074~1080
```

- 空格分隔关键词。
- 单引号或双引号可以把带空格文本作为一个关键词。
- 一个 `start~end` 年份范围会被识别为时间条件。
- 事件区间与搜索范围相交即匹配。
- 结果按事件开始时间排序。

当前不提供布尔表达式、正则表达式或模糊拼写纠正，避免产生难以解释的结果。

## 搜索作用域

- 全部图层
- 可见图层
- 当前图层
- 只搜 `primary`
- 只搜 `context` / `background`

“无关事件变暗”保留上下文但降低非匹配事件；“只显示匹配”会从当前绘制中移除非匹配事件。清除搜索会恢复原视图和显示状态。

## 过滤器

过滤条件与搜索关键词叠加：

- 图层组
- 图层角色
- 标签
- 地点
- 人物
- 起止年份
- 当前视窗

人物、地点和标签输入多个值时采用 AND；每个值可以部分匹配元数据中的实际文本。

## 快捷键

| 快捷键 | 行为 |
| --- | --- |
| `/` | 聚焦全局搜索 |
| `Esc` | 清除搜索；无搜索时关闭菜单 |
| `Ctrl/Cmd + Z` | 撤销事件、图层、分组或项目编辑 |
| `Ctrl/Cmd + Shift + Z` | 重做 |
| `0` | 重置视图到完整数据范围 |
| `←` / `→` | 平移当前视图；Shift 加速 |
| `+` / `-` | 围绕画布中心缩放 |
| `Enter` | 画布聚焦时在中心固定探针 A |
| `Shift + Enter` | 固定探针 B |
| `Delete` | 删除选中事件（先确认） |

Canvas 也支持拖拽平移、滚轮围绕鼠标缩放、单指方向锁定平移 / 垂直浏览、双指缩放和双击重置。
